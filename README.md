## About

This is the official website for **Angel Eyes by Di**, a lash extension artist based in New Jersey. The site was designed to match her brand — soft lavender tones, elegant typography, and sparkle animations throughout.

---

## Features

- 💜 Full services & pricing menu with tabbed navigation
- 📸 Photo gallery showcasing her lash work
- 📋 Appointment policies section
- 📅 Direct booking link to GlossGenius
- ✨ Floating sparkle canvas animation + mouse trail
- 📱 Fully responsive for mobile and desktop
- 🌙 Smooth scroll reveal animations on every section

---

## Pages / Sections

- **Hero** — intro with booking CTA and availability badge
- **About** — Di's story in her own words
- **Services** — Full Sets, Fills, and Other services with pricing
- **Gallery** — portfolio of her lash work
- **Policies** — appointment rules and expectations
- **Contact** — Instagram, phone, and booking link

---

## Built With

- HTML5
- CSS3 (Grid, Flexbox, Animations, Canvas API)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond + Jost)
- Hosted on GitHub Pages
